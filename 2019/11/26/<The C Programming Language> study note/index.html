<The C Programming Language> study note

main function is the program entry point.

Variables

All variables must be declared before use.
Variable declaration: type stated && memory allocated.

A char is just a small integer.
A character written between single quotes represents an interger value equal to the numerical value of the character in the machine’s character set.
So chars may be freely used in arithmetic expressions.

Functions

A function provides a convenient way to emcapsulate some computations, which can then be used without worrying about its implementation.

In C, all function arguments are passed “by value”. This means that the called function is given the values of its arguments in temporary variables rather than the originals.
The story is different for arrays. When the name of an array is used as an argument, the value passed to the function is the location or address of the beginning of the array – there is no copying of array elements. By subscripting this value, the function can access and alter any element of the array.
